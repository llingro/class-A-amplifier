兩級主動偏置 Class A 放大器與主動恆流源負載設計

Two-Stage Active-Biased Class A Amplifier with Active Current Sink & Negative Feedback

本專案是一個在 LTspice 中精心設計與調校的高性能、高穩定度 Class A（甲類）類比音頻放大器。電路從最初的直流偏置崩潰、正回授發散，經歷了直流隔離、交流負回授引入，並最終升級至「多路主動式電流鏡偏置」與「主動式恆流源負載」，達到了極佳的線性度、超低失真度與高度對稱的動態電壓擺幅。

一、 電路基本規格 (Specifications)

電源電壓 ($V_{CC}$)：$12\text{ V}$

中頻電壓增益 ($A_v$)：$\approx 46\text{ dB}$（電壓放大倍數約 $200$ 倍）

輸出級靜態工作電流 ($I_{CQ4}$)：$\approx 1.01\text{ mA}$（超低靜態功耗設計）

輸出靜態直流偏置點 ($V_{EQ4}$)：$5.65\text{ V}$（極度接近黃金中點 $6\text{ V}$，提供最大對稱輸出擺幅）

輸入訊號容忍度：在閉迴路負回授下，可完美放大 $10\text{ mV}$ 級別之交流訊號不切頂。

二、 電路架構與設計特點 (Circuit Topology)

本電路融合了多個經典類比集成電路（Analog IC）的設計精髓：

                    [ 12V Supply ]
                         │
         ┌───────────────┼──────────────┐
         ▼ (Bias)        ▼ (Gain)       ▼ (Output)
       [ R1 ]          [ R2 ]         [ Q4 (CC) ] ───( C4 )───► [ Output R7 ]
         │               │              │
       [ Q2 ] ──(Bias)──[ Q3 (CE) ]     │
       (Mirror)          │              ▼ (Active Load)
         │             ( C2 )         [ Q5 (Current Sink) ]
         ▼               ▼              ▼
       [ GND ]         [ GND ]        [ GND ]


第一級：共射極（CE）放大器 ($Q_3$)

提供電路主要的電壓增益（Voltage Gain）。

射極配有旁路電容 $C_2 = 100\,\mu\text{F}$，在交流訊號下將射極接地，最大化開迴路增益。

第二級：共集極（CC）隨耦放大器 ($Q_4$)

作為輸出緩衝級，提供極高的輸入阻抗與極低的輸出阻抗，用於驅動負載 $R_7$ ($500\,\Omega$)。

主動式多路電流鏡偏置 ($Q_1, Q_2, Q_5$)

$Q_1, Q_2$ 組成標準電流鏡，為第一級 $Q_3$ 提供高阻抗的主動偏置電流。

主動式恆流源負載 ($Q_5$)：將 $Q_5$ 的基極接回電流鏡基準源。$Q_5$ 取代了傳統 Class A 輸出級效率極低的射極電阻，在 $Q_4$ 的射極端提供穩定的 $1\text{ mA}$ 主動拉電流（Pull-down）。這徹底解決了下半週放電緩慢、容易產生不對稱切頂（Clipping）的缺點。

交直流隔離式跨級負回授 ($C_3, R_6$)

從 $Q_4$ 射極（輸出端）透過 $C_3 = 2.2\,\mu\text{F}$ 與 $R_6 = 10\text{ k}\Omega$ 接回 $Q_3$ 基極（輸入端）。

直流隔離（DC Blocking）：$C_3$ 阻斷了直流電壓，使輸出級的直流電壓不干擾輸入級偏置。

交流負回授（AC Feedback）：大幅降低非線性失真，穩定交流增益，並顯著拓寬放大器頻寬。

三、 高頻性能考慮 (High-Frequency Consideration)

為了模擬電晶體在現實物理世界中的高頻極限與密勒效應（Miller Effect），本專案摒棄了過於理想化的簡易 SPICE 模型，採用了包含半導體寄生接面電容與傳遞時間的完整高頻 BJT 模型：

.model MYNPN NPN(Bf=150 Vaf=100 Cje=25p Cjc=8p Rb=100 Tf=0.3n)


高頻參數解析：

Cje = 25p：發射極接面電容，主要決定電晶體輸入端寄生電容 $C_\pi$。

Cjc = 8p：集電極接面電容，決定了關鍵的基極-集極電容 $C_\mu$。由於 $Q_3$ 具有高電壓增益，該電容會因密勒效應而被放大 $(1 + A_v)$ 倍，是高頻截止頻率（$f_H$）的主要決定因素。

Rb = 100：基極體電阻，與寄生電容交互作用，在高頻形成極點。

Tf = 0.3n：順向過渡時間（Forward Transit Time），限制了載子穿過基極的最快物理速度。

四、 如何在 LTspice 中運行模擬

1. 直流工作點分析 (.op)

在 LTspice 中點擊 Run，觀察直流節點數值：

確認 $V(n004)$（$Q_3$ 基極）維持在約 $2.04\text{ V}$。

確認 $V(n009)$（$Q_4$ 射極）穩定在約 $5.65\text{ V}$。

確認 $Q_4$ 的集極直流電流 $I_c(Q4) \approx 1\text{ mA}$。

2. 時域瞬態分析 (.tran)

使用以下指令觀察放大後的正弦波形：

.tran 0.01


輸入信號 $V_3$：設定為 SINE(0 0.01 1k)（即 $10\text{ mV}$ 振幅、$1\text{ kHz}$ 正弦波）。

輸出波形：在 $V(n009)$ 或 $C_4$ 右側觀測。波形應呈現無失真、不切頂、上下完美對稱的 Class A 放大特性。

3. 交流頻率響應分析 (.ac)

使用以下指令觀測電路的帶寬與高頻衰減特性：

.ac dec 20 1 100meg


中頻段：增益維持在 $46\text{ dB}$。

低頻截止：由 $C_1, C_2, C_3, C_4$ 決定，增益在 $100\text{ Hz}$ 以下開始衰減。

高頻截止：歸功於高頻寄生參數模型的啟用，您將能在高頻（數 $\text{MHz}$ 處）清晰看到增益陡峭下滑（Roll-off）與相位滯後，這是最真實的電路高頻物理特性呈現。

五、 設計演進歷程 (Evolution History)

V1.0：最初設計。直流未隔离，Q4 電流暴增至 $39.92\text{ mA}$，電路嚴重發熱，波形失真。

V2.0：嘗試回授，但誤接為「正回授」，直流點不穩且波形隨時間向上發散漂移。

V3.0：修正回授方向至輸入基極，成功隔離交直流，但 $R_5$ ($30\,\Omega$) 過小導致輸出級 $101.69\text{ mA}$ 大電流。

V4.0：將 $R_5$ 改為 $1\text{ k}\Omega$，電流降至 $6\text{ mA}$，直流點救回，但大訊號時負半週仍有截止切頂。

V5.0 (最終版)：引入 $Q_5$ 主動式恆流源負載，電流完美控制在 $1.01\text{ mA}$，同時兼顧省電與極致的對稱放大波形。
